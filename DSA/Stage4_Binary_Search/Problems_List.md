### 💡 Key Learnings

- Find the left boundary first.
- Find the right boundary second.

# 📌 Important Observations

- Binary Search is about reducing the search space.
- Boundary problems require continuing the search after finding the target.
- Answer Binary Search works only when the search space is monotonic.
- Always verify the search space before writing code.
