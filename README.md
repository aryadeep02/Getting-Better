# Becoming a Better Engineer

Hi, I'm Aryadeep Varshney, an aspiring software engineer currently working as an entry-level engineer while also freelancing on the side.

Right now, I’m focused on improving myself as a developer by learning full-stack development, data structures & algorithms, backend engineering, and system design while consistently building projects and sharpening my problem-solving skills.

As a recent graduate from a Tier-3 university, I know the path might be a little harder and maybe a little later than others — but definitely not too late to build something meaningful.

This repository is a documentation of my journey — partly for my future self, and partly for every engineer who feels like they started late.

I’ve realized that improving a little every day matters far more than constantly worrying about being behind. The goal here is simple: keep learning, keep building, and keep moving forward consistently.

I graduated recently and decided to seriously focus on becoming a strong engineer instead of endlessly consuming tutorials and roadmaps.

Right now, my main focus areas are:
- Data Structures & Algorithms
- Java Full-Stack Development
- Spring Boot
- SQL & Databases
- CS Fundamentals
- Building projects consistently

I'm currently solving the Striver SDE Sheet alongside learning backend engineering step by step.

The purpose of this repository is simple:
- track progress
- stay consistent
- document learnings
- build in public for my future self

No fake productivity.  
No overnight success story.  
Just consistent work, learning from mistakes, and improving every day.
I’ll also be documenting my mistakes along the way so others starting from a similar place can learn from them too.
