->Learned the difference between directories, packages, and modules in IntelliJ IDEA:

* Directories are normal folders used for organizing files.
* Packages are Java-specific structures used to organize classes and maintain clean code architecture.
* Modules are separate mini-projects inside a larger project, each having its own dependencies and configurations.
